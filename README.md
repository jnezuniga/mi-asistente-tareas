# 📝 Mi Asistente de Tareas v2.0

Aplicación web para organizar tus tareas diarias con categorías, prioridades y edición completa.

![Versión](https://img.shields.io/badge/versión-2.0-blue)
![Estado](https://img.shields.io/badge/estado-activo-success)

---

## ✨ Características

### Funcionalidades principales:
- ✅ **Crear tareas** - Añade tareas con descripción, categoría y prioridad
- ✏️ **Editar tareas** - Modifica cualquier aspecto de tus tareas
- 🏷️ **Categorías** - Organiza en: Trabajo, Personal, Urgente, Otros
- ⭐ **Prioridades** - Define importancia: Alta, Media, Baja
- 🔍 **Filtros** - Visualiza tareas por categoría
- ✓ **Marcar completadas** - Seguimiento de progreso
- 📊 **Estadísticas** - Total, pendientes y completadas
- 💾 **Persistencia** - Tus tareas se guardan localmente
- 📱 **Responsive** - Funciona en celular, tablet y escritorio

---

## 🚀 Uso

1. **Accede a la aplicación** en: [tu-url-aqui.github.io/mi-asistente-tareas/]
2. **Añade tu primera tarea** escribiendo en el campo superior
3. **Selecciona categoría y prioridad** antes de añadir
4. **Click en el texto** de cualquier tarea para editarla
5. **Usa los filtros** para ver tareas por categoría
6. **Marca como completada** con el checkbox

## 📱 Instalación como App

### Android:
1. Abre la app en Chrome
2. Menú (⋮) → "Añadir a pantalla de inicio"
3. ¡Listo! Tendrás un icono como app nativa

### iOS:
1. Abre la app en Safari
2. Botón Compartir (□↑) → "Añadir a pantalla de inicio"
3. ¡Aparecerá con el icono 📝!
